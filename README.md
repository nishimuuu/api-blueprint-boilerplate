# api-blueprint-boilerplate


## Requirements
- npm (3.8.3)
- gulp


## Install
1. Run `npm install -D`

## How to use
1. Write document formatted by api-blueprint to /doc/md/
2. Run `gulp`
3. [API Document]   
Access http://localhost:10000    
[API mock]  
Access http://localhost:3000  

## How to change path, port..

Modify gulp.yml
